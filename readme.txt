== Description ==\
\
To open hyperlinks as a popup. Advantage of using this plugin is not to focus from parent page, more like a preview/demo of the link.\
\
Note: for IE, it will open hyperlinks in new window.\
\
\
== Installation ==\
\
1. Download plugin\
2. Activate the plugin through the 'Plugins' menu in WordPress\
3. add rel='pop' to the tag e.g., <a href="link_url" rel="pop"></a>\
4. The title of the popup is the title of hyperlink if provided, otherwise the link url\
\
== Frequently Asked Questions ==\
\
= Will this work in all browsers and all doctypes? =\
\
Works in non IE browsers, but for IE, it will open hyperlinks in new window. \
Works in strict, transitional DTD doctype.\
